# 📚 Study Chatbot API

This is a FastAPI-based chatbot API designed to answer study-related questions.  
It provides intelligent responses through a simple REST API interface.

---

## 🚀 Features

- REST API built using FastAPI
- Study-focused chatbot responses
- JSON-based API responses
- Lightweight and easy to integrate
- Easy to run locally

---

## 🛠 Tech Stack

- Python
- FastAPI
- Uvicorn

---

## ▶️ How to Run the Project

### 1️⃣ Clone the Repository

git clone https://github.com/bvsangeetha88-rgb/study_chatbot.git

### 2️⃣ Navigate to the Folder

cd study_chatbot

### 3️⃣ Create Virtual Environment

python -m venv venv

### 4️⃣ Activate Virtual Environment

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

### 5️⃣ Install Dependencies

pip install -r requirements.txt

### 6️⃣ Run the Server

uvicorn app:app --reload

### 7️⃣ Open in Browser

http://127.0.0.1:8000
